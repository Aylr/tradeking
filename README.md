tradeking-apis
==============

Wrappers for TradeKing's API calls.


requirements
============

requests-oauth https://github.com/maraujop/requests-oauth


setup
=====

set environment variables:
  TK_CONSUMER_KEY
  TK_CONSUMER_SECRET
  TK_OAUTH_TOKEN
  TK_OAUTH_SECRET
to the values given by tradeking or pass them in explicitly.
